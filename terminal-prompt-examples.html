macOS Terminal Prompt Examples with Git Branch Display

1. Powerlevel10k (Recommended)
Features:
  • Fast rendering and extensive customization
  • Interactive configuration wizard
  • Git status with visual indicators
  • Works with or without Nerd Fonts
olindo@MacBook-Pro ~/prj/k8s-lab ⎇ main ✓
❯ git status
olindo@MacBook-Pro ~/prj/k8s-lab ⎇ feature/new-helm ⚡3 ✓
❯ helm install...
Installation:
git clone --depth=1 https://github.com/romkatv/powerlevel10k.git ${ZSH_CUSTOM:-$HOME/.oh-my-zsh/custom}/themes/powerlevel10k
ZSH_THEME="powerlevel10k/powerlevel10k" # in ~/.zshrc
p10k configure # run configuration wizard
2. Starship (Cross-Shell)
Features:
  • Works with Bash, Zsh, Fish, PowerShell
  • Fast and minimal by default
  • Highly configurable with TOML
  • Great documentation
~/prj/k8s-lab git:(main) ❯
kubectl get pods
~/prj/k8s-lab git:(feature/auth) ↑2 ❯
git push origin feature/auth
Installation:
brew install starship
echo 'eval "$(starship init zsh)"' >> ~/.zshrc
3. Oh My Zsh - Agnoster Theme
Features:
  • Built into Oh My Zsh
  • Powerline-style arrows
  • Clear git status indication
  • Requires Powerline fonts
olindo⮀~/prj/k8s-lab⮀ main⮀
olindo⮀~/prj/k8s-lab⮀ feature/helm ±⮀
Installation:
ZSH_THEME="agnoster" # in ~/.zshrc
# Requires Powerline fonts or Nerd Fonts
4. Spaceship Prompt
Features:
  • Modular sections
  • Asynchronous rendering
  • Extensive customization options
  • Clean, modern design
olindo in k8s-lab on main
🚀 microk8s status
olindo in k8s-lab on feature/servarr [!]
🚀 helm template servarr
Installation:
git clone https://github.com/spaceship-prompt/spaceship-prompt.git "$ZSH_CUSTOM/themes/spaceship-prompt" --depth=1
ln -s "$ZSH_CUSTOM/themes/spaceship-prompt/spaceship.zsh-theme" "$ZSH_CUSTOM/themes/spaceship.zsh-theme"
ZSH_THEME="spaceship" # in ~/.zshrc
5. Pure Prompt (Minimalist)
Features:
  • Minimal and fast
  • Two-line prompt
  • Async git status
  • No dependencies
~/prj/k8s-lab main
❯ kubectl apply -f servarr/
~/prj/k8s-lab feature/new-chart ⚡
❯ git add .
Installation:
npm install --global pure-prompt
# Add to ~/.zshrc: autoload -U promptinit; promptinit; prompt pure
6. Oh My Zsh - Robbyrussell (Default)
Features:
  • Default Oh My Zsh theme
  • Simple and clean
  • No special fonts required
  • Basic git status
olindo@MacBook-Pro k8s-lab git:(main) $ helm list
olindo@MacBook-Pro k8s-lab git:(feature/updates ✗) $ git status
Installation:
ZSH_THEME="robbyrussell" # in ~/.zshrc (default)
