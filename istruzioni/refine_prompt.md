# Prompt per Raffinare un Sotto-Piano

Copia e incolla questo testo per modificare in dettaglio uno specifico piano di migrazione.

---

```text
Sistema, voglio raffinare o modificare i dettagli di uno specifico piano di migrazione per il cluster Talos.

Per favore:
1. Leggi la strategia generale in: `istruzioni/storage_strategy.md`.
2. Chiedimi QUALE dei 4 piani voglio modificare (01_talos_setup, 02_storage_setup, 03_data_recovery, 04_k8s_manifests).
3. Leggi il file corrispondente che si trova in `istruzioni/migration/`.
4. Attendi le mie specifiche istruzioni su cosa cambiare (es. "Voglio cambiare la dimensione dei dischi", "Voglio aggiungere un passaggio", ecc.) e aggiorna il file del piano.
```
